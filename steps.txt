Folder structure : 


PhishingDetector/               Main project folder
│
├── app.py                      # Main Flask backend code
├── requirements.txt            # List of Python libraries (Flask, scikit-learn, etc.)
│
├── templates/                  # FOLDER: Contains HTML
│   └── index.html              # Main frontend page
│
├── static/                     # FOLDER: Contains assets
│   ├── style.css               # Design/CSS file
│   └── script.js               # Frontend logic/JavaScript file
│
├── text_phish/                 # FOLDER: Contains Text ML files
│   ├── text_model.pkl          # Trained Spam Classifier
│   └── vectorizer.pkl          # The TfidfVectorizer used during training
│
├── url_phish/                  # FOLDER: Contains URL ML files
│   └── url_analysis_model.joblib  # Trained URL Phishing Model
│
└── venv/                       # FOLDER: Virtual Environment (Automatic)
    └── ...                     # (Created after run 'python -m venv venv' or 'py -m venv venv')
